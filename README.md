# My Portfolio

A modern, responsive portfolio website built with **React.js**, **Tailwind CSS**, and **Supabase**.

## 🚀 Features
- **Serverless Architecture**: Powered by Supabase for database and storage.
- **Dynamic Content**: Manage projects, skills, and experience through an Admin Dashboard.
- **Modern UI**: Sleek design with dark mode support and smooth animations (Framer Motion).
- **Responsive**: Fully optimized for all screen sizes.
- **Secure**: Implements Row Level Security (RLS) for data protection.

## 🛠️ Tech Stack
- **Frontend**: React.js, Tailwind CSS, Framer Motion
- **Backend/Database**: Supabase (PostgreSQL)
- **Storage**: Supabase Storage for assets and resumes

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Chidvilas-264/MyPortfolio.git
   ```

2. **Install dependencies**:
   ```bash
   cd frontend
   npm install
   ```

3. **Environment Variables**:
   Create a `.env` file in the `frontend` directory and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the app**:
   ```bash
   npm run dev
   ```
